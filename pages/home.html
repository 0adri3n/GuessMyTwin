How to play?

  1. Create a room or join an existing room via URL
  2. Wait for a second player to join
  3. The host chooses the character set and starts the game
  4. Guess your opponent's character!

Create a Mod

To add your own character set for the game, follow these steps:

  1. Click the "Create a Mod" button.
  2. Select multiple image files (supported formats: jpg, png, gif, webp).
  3. Choose a folder/name when prompted ‚Äî the app will create a folder for the mod.
  4. The application copies the selected images into the new folder and generates a .json configuration file describing the characters so the game can use them.

After creation you can import the mod from the lobby and use it in your next game.
